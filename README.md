# PuzzleForge

**Free KDP Puzzle Book Generator — The BookBolt Alternative**

🔗 [puzzleforge.com](https://puzzleforge.com) · Pure HTML/CSS/JS · No build step · No backend

## Features

- **6 Puzzle Types** — Word Search, Word Scramble, Crossword, Sudoku, Hangman, Maze
- **Bulk Generation** — 1 to 50 unique puzzles per session
- **Answer Keys** — Auto-generated, sorted to back of PDF
- **PDF & PNG Export** — Print-ready at KDP page sizes
- **BYOK AI** — Bring your own API key (Gemini, OpenAI, Claude, Cohere, Mistral)
- **600+ Built-in Words** — 20 curated categories, no AI needed
- **Hangman Clue Database** — 60+ words with descriptive clues built in
- **KDP Page Sizes** — 8.5×11", 6×9", 5×8"
- **SEO Optimized** — Schema markup, FAQPage, meta tags, sitemap
- **AdSense Ready** — 3 placeholder slots pre-configured
- **Legal Pages** — Privacy Policy, About, Disclaimer, Contact built in

## Deploy

1. Rename `puzzleforge.html` → `index.html`
2. Upload all files to GitHub repo
3. Connect repo to Cloudflare Pages
4. Set build output to `/` (root — no build needed)
5. Point your domain in Cloudflare
6. Submit `sitemap.xml` in Google Search Console

## AdSense

Replace the three `ad-placeholder` divs in `index.html` with your actual AdSense code after approval. Slots are labeled with comments.
